# Mentiva

Your vision board, with a brain. Create vision boards and use AI to clarify goals and turn inspiration into action.

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** (dark sage green + gold theme)
- **TypeScript**

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

- `/` — Landing
- `/login` — Sign in
- `/dashboard` — Your boards
- `/upload` — Upload vision board
- `/chat` — AI chat about your board

## Build

```bash
npm run build
npm start
```
